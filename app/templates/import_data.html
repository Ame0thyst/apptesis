Panduan Import

  • Gunakan format file .xlsx (Excel) atau .csv.
  • Kolom wajib: Nama, NISN, Kelas.
  • NISN akan digunakan sebagai Username dan Password default.
  • Kolom opsional: Role, Username, Password. Jika tidak diisi, sistem memakai default.
  • Nilai Role untuk import ini adalah siswa.
  • Pastikan tidak ada NISN ganda di dalam file atau database.
  • Gunakan tipe Text untuk kolom NISN agar angka nol di depan tidak hilang.
